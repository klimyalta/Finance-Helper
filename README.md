# Finance Helper — Мобильный финансовый помощник

PWA-подобное веб-приложение для учёта личных финансов в стиле мобильного: расходы по категориям, статистика, цели накопления и напоминания. Всё работает в браузере, данные хранятся локально.

**Основные возможности:**
- Добавление расходов с категориями, суммой, комментарием и датой  
- Статистика по категориям с круговой диаграммой (чистый CSS)  
- Цели накопления с прогресс-барами, внесением средств и комментариями  
- Напоминания с датой/временем и удалением  
- Баланс с ручным редактированием  
- Тёмная тема, модальные окна, bottom-навигация, уведомления (notify)  
- Полностью клиентский — без сервера, данные в localStorage  
- Responsive дизайн: имитация мобильного экрана, полноэкранный режим на телефоне  

**Технологии:**
- HTML5 (структура с секциями и модалками)  
- CSS (flex/grid, градиенты, clip-path для pie-chart, медиа-запросы для mobile)  
- Vanilla JavaScript (утилиты, рендер, события, localStorage)  
- Font Awesome (иконки) + Google Fonts (Inter)
